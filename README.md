# CalculadoraSimple
Calculadora que da opcion de elegir entre:
</br>
°Suma
°Resta
°Multiplicación
°División
Si se escribe una palabra o no se escribe un numero donde es debido,
detecta el error y pide que vuelva a escribir lo que desea
